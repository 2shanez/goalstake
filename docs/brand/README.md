## Design Direction

**Vaada** = "promise" in Hindi

Inspired by:
- **Aave** — Simple, memorable non-English word
- **Pinky promise** — Universal symbol of commitment
- Modern crypto logos (clean, geometric, scales well)

### Logo Requirements
- Works at 32px (favicon) and 512px (app icon)
- Recognizable in single color
- No fine detail (survives compression)
- Rounded strokes preferred
